语法

Enumerate/Select

  • {V / Nの} かたわら: ー边...ー边...
    彼かれは会社かいしゃ勤つとめのかたわら、福祉ふくし活動かつどうに積極せっきょく的てきに取とり組くんでいる。

  • {V} か {Vない} か: 是...还是...
    {小句} かどうか: 是否...
    留学に行くか行かないかを迷っています。
    彼がこのプレゼントを 喜 ぶかどうかわかりません。

  • {V意志} か {V} まいか : ...还是不...
    彼かれは夏休なつやすみに国くにへ帰かえるか帰かえるまいかと悩なやんでいるらしい。

  • {V / AI / AN / N} というか / といおうか {V / AI / AN / N} というか: 该说是...好呢，还是...好呢，总之...
    一人ひとりであんな危険きけんな場所ばしょへ行いくとは、無茶むちゃというか、無知むちというか、とにかく私わたしには理解りかいできない。

  • {V / AI / AN / N} やれ {N} やれ {N}: 什么...啦，什么...啦
    今月こんげつはやれ子こどもの日ひだやれ娘むすめの誕生たんじょう日びだと、お祝いわいが重かさなる。

Thought

  • {V}にたえない: 忍耐不了, 耐[た]える: 忍受;值得;胜任
    この番組はひどくて見るにたえない。

  • {N / AI / ANな　/　Nの} ものと思われる: 认为是... 犯人はんにんは海外かいがいへ逃亡とうぼうしたものと思おもわれる。

Possibility

  • {V} ベくもない(する → すベくもない/するベくもない): 不可能~, 可く[べく]
    主力しゅりょくが抜ぬけているので、優勝ゆうしょうは望のぞむベくもない。

Advise

  • {V / N} に如くはない: 不如..., 如し
    逃にげるにしくはなし。

Causality

  • {Vば / Aば / Nであれば} こそ：正因为~
    臆病おくびょうであればこそ、用心深ようじんぶかくなり、危険きけんを避さけることができるのだ。

  • {V / AI / ANな} 羽目になる / 羽目に陥る: 因为...陷入...境地, 羽目はめ：困境, 陥るおちる：陷入
    うっかりしてデータを消けしてしまったので、もう一度いちど入力にゅうりょくする羽目はめになった。

  • {V / AI / AN / Nの} ことだし: 因为...(陈述其中一个理由)
    雨あめも止やんだことだし、そろそろ帰かえりましょうか。

  • {V / AI / AN / N} {V / Vた} {の / ん / N} じゃあるまいし / ではあるまいし: 因为不是...(所以没必要)
    期末きまつレポートといっても別べつに外国がいこく語ごで書かくんじゃあるまいし、そんなに心配しんぱいしなくていいよ。

  • {V / AI / AN / Nの} こととて: 因为...所以...
    昼間ひるまはにぎやかなこの町まちも、早朝そうちょうのこととてあたりに人影ひとかげはなかった。

Reference

  • {N}に照らして: 依照~, 照[て]らす: 对照,照耀
    法律ほうりつに照てらして処分しょぶんする。

  • {N} につきまして： 关于~ (...について・...にとって・...において・...として更加礼貌的说法)
    発売はつばいの時期じきにつきましては、現在げんざい検討けんとう中ちゅうでごさいます。

  • {N}（で）すら: 甚至~(「すら」前面的助词「で」可以省略)
    そんなことは子どもですら知っている。

  • {N}ったら: 说起~(和熟人说)
         ってば(口语)
    このパソコンったら、しょっちゅうフリーズするんだ

  • {N} にかかったら/にかかっては/にかかると: 说到...彼かれの毒舌どくぜつにかかっては、誰だれも太刀打たちうちできない。

Exclude

  • {N} はそっちのけで / をそっちのけで: 不顾~(把本应该做的亊情放在ー边，而去做其他事情)
    生徒せいとたちは教師きょうしの注意ちゅういをそっちのけで夢中むちゅうだ。

Comparison

  • もさることながら: 相比前项后项更加~, ~自不必说 ~更是如此
    ごみを減へらすためには、市しや町まちの取とり組くみもさることながら、個人こじんの心こころがけもやはり大切たいせつだ。

  • {V} くらいなら：与其...不如...
    15000円えんも払はらうくらいなら、新あたらしいのを買かったほうがいいね。

  • {V} まし：比...好
    不景気ふけいきなんだから、仕事しごとがあるだけましだと思おもわなきゃいけないんだぞ。

Infer

  • {V（が）/Vた（が）/ Nの} ごとき / ごとく / ごとし： 像~, 如ごとく (与「〜ような」〜ように」「〜ようだ」用法相同) 「反省はんせい」とは、あたかも他人たにんを眺ながめるがごとく 自みずからを客観きゃっかん的てきに観察かんさつして行おこなう精神せいしん活動かつどうである。

タイミング

  • {V连用} たて: 刚~, 立て[たて]
    このベンチ、ペンキぬりたてだって。

  • {V / Vた} が早いか: 刚刚...就...
    子供こどもたちは動物園どうぶつえんに着つくが早はやいかお菓子かしを食たべだした。

  • {V} なり: 一...就...
    娘むすめは家いえに帰かえってくるなり、泣なき出だした。

顺接

  • {N} にして {N}: ~的同时, 既是~ 又是~
    彼は医者いしゃにして作家さっかでもある。

限定/量/程度

  • [Vて / AIて / ANで] しかた(が)ない / しょうがない / しようがない：(没办法了)非常~
    主语是他人时末尾加上らしい / そうだ / ようだ
    夕ベ徹夜したので，眠くてしょうがない。

  • [Vて / AIて / ANで] たまらない：(忍受不了)非常~, 堪たまる: 忍受
    昨日の歌の練習では何度も間違えてみんなに笑われ，恥ずかしくてたまらなかった。

  • {V / AI / AN / N} といったら（ありはし）ない: 非常~
                              ったらない / ったらありゃしない: 非常~(口语形式)
    結婚けっこんが決きまったときの彼女かのじょの喜よろこびようったらありゃしなかった。

  • {V / ANな} のなんのって: 非常~
    虫歯むしばを抜ぬいたら、痛いたいのなんのって。何なにも食たべられないんだ。

  • {V / AI / AN / N} にもほどがある: ...也应该有个度(固定用法「冗談にもほどがある」)
    先さきに対たいして、あんな失礼しつれいなことを言いうなんて、無神経むしんけいにもほどがある。

  • 今ひとつ...(ない): 感觉还有欠缺
    この力ちからパンは使つかい勝手がってはよいが、デザインが今いまひとつだ。

  • {V / Vた} までだ / までのことだ: 只是...而已
    念ねんのため、確認かくにんするまでのことだ。

转折

  • {N} たりとも： 即使也
    猫の子一匹いっぴきたりとも、ここを通とおらせないぞ。

  • {V / AI / AN / N} とまではいかないが / とまではいかないとしても: 虽然达不到...的程度，但是（至少）...
    毎日まいにちとまではいかないが、週しゅうに 2、3回かいは掃除そうじをしようと思おもう。

  • {V连用 / Vない / AI / AN / N} ながらも: 虽然...但是...
    彼かれは、貧まずしいながらも、温あたたかい家庭かていで育そだった。

  • {Nだ / 动た} とて: 即使...也...(常与「たとえ」、「どんなに」、「いくら」等词一起使用)
    たとえ病気びょうきだとて試験しけんに欠席けっせきしてはいけない。

  • {V / AI / AN / N} とはいえ: 虽然...但是...
    春はるになったとはいえ、まだまだ寒さむい日ひが続つづいている。

被动

  • 直接受身文: を/に格的V (41课)

    • 动作主 + に
      李さんは部長にほめられました。
    • 动作主不出现
      ビルが 建てられました。
    • 动作主 + によって: 由…做
      创造发明:作る、設計する、建てる、発明する、書く……
      この本は 山田先生によって 娘に 書かれました
    • 动作主 + から: 由…传递
      物品的移动:与える、渡す、贈る
      金メダルが林先生から太郎に渡されました
      林先生から太郎に金メダルが渡されました(生活中常用)

  • 間接受身文/迷惑受身文(被害): 主语是受害人(41课)
    陳さんは 飼っていた小鳥に 逃げられました。

  • 持ち主受身文: 主语是物主, 所有物承受动作(41课)
    馬さんは 森さんに カメラを 壊されました。

  • 尊敬
    部長は帰られました。

使役

  • ~は ~を/に {自Vさせる}(43课)
李さんが出張する。(自動詞)
部長が李さんに/を出張させる。

Info

  • 只能用を / を和に皆可:

~を

  • 只能用に: 已经有を格

親は 子供に/を 步かせる。
親は 子供に 步道を 步かせる。

  • ~は ~に ~を{他Vさせる}(43课)
森さんが歌を歌う。(他動詞)
部長が森さんに歌を歌わせる。
  • 強制/容認/原因/責任

部長が 李さんを 出張させる。(強制·指示)
子供を游ぼせた。(許可·容認)
自由に公園で志望校の合格は 両親を とても喜ばせた。(原因)
誰が日本のコロナ福を悪化させたのか。(責任)

  • 謙譲

私から説明させていただきます。(謙譲)

使役被动

李さんが出張する。(自動詞)
部長が李さんに/を出張させる。(使役)
李さんが部長に出張させられる。(使役受身)
森さんが歌を歌う。(他動詞)
部長が森さんに歌を歌わせる。(使役)
森さんが部長に歌を歌わされた。(使役受身)

条件(と/たら/ば型/なら)

  • {Vば} よかった：要是（不）~的话就好了（「〜なければよかった」在口语中可以说成「〜なきゃよかった」）
    彼女かのじょはわたしの話はなしに傷きずついたみたい。あんなことを言いわなきゃよかった。

  • {Vば} と思う：如果~就好了(省略了幸い/ありがたい)
    皆みなさんのご意見いけんをお聞きかせくださればと思おもいます。

  • {动辞书形/名} とあっては: 如果...
    上司じょうしの指示しじとあっては、仕事しごとの内容ないようが気きに入いらなくてもやらなければならない。

  • いざとなると/いざとなれば/いざとなったら: 一旦有情况...就...(惯用短语，可单独使用)
    何度なんども練習れんしゅうしてきたが、いざとなるとなかなか思おもうとおりに進すすまない。

  • {N / 简体句+か} にかかっている: 取决于...
    君きみたちが成功せいこうするかどうかは、与あたえられたチャンスをどう使つかうかにかかっている。

动词

授受动词

Nを授受 (物品)Vて授受 (动作)
あげる我给别人(我→他,施恩)私／母は李さんに花をあげました。私／母は李さんに花を買ってあげました。
客观描述森さんは小野さんに花をあげました。森さんは小野さんに花を買ってあげました。
くれる别人给我(他→我,受恩)李さんは私／母に花をくれました。李さんは私／母に花を買ってくれました。
もらう别人给我 / 我向别人要(他→我,受恩,更委婉更常用)私／母は李さんに／から花をもらいました。私／母は李さんに花を買ってもらいました。
客观描述小野さんは森さんに／から花をもらいました。小野さんは森さんに花を買ってもらいました。

辅助动词

  • {V连用} そびれる: 错失机会...
    今日きょうは生憎あいにくの曇くもり空そらで中秋ちゅうしゅうの名月めいげつを曇くもりそびれた。

限る

  • {N} に限って / に限り：唯独~, 只有~
    毎週火曜日に限って、映画は半額はんがくになります。

  • {V / Vない / ANな / Nの} 限り：只要(不)~ 就~
    生きているかぎり、うれしいことも悲しいことも必ずある。

  • {Vる / Vた} 限り / 限りで(は)：限定范围, 根据~
    わたしの見る限りでは、彼は信頼できる人物だ。
    私もできる限りのことをしたい。

  • {V / Vない / N} に限る: 最好是~(相当于「〜のが一番いい」)
    疲れたときは、寝るに限ります。

  • {Vる / V可能型 / Nの} 限り：尽量~
    できるかぎりの努力れんりょくはした。あとは結果けっかを待つのみだ。

  • {简体句} とは限らない: 不见得~, 未必~
    先生だからといって、何でも知っているとは限らない。

  • {N} に限らず: 不仅~而且
    今日のセミナは就職活動中の学生に限らず、新入生の私たちにも役に立った。

  • {N} 限って： 偏偏~(运气不好)
    急いそいでいるときに限かぎって、バスが来こない。

わけ

  • {V / ANな / AI / Nの} わけがない / わけはない：不可能~, 不会~
    お酒の好きな田中さんが来るんだから，これだけで足りるわけがない。

  • [V / ANな / AI / Nの] わけではない / わけでもない：并不是~, 并非~
    最近の子どもはテレビゲームばかりしているようだが，必ずしも外で遊ばないわけではない。

  • [V / Vない] わけにはいかない / わけにもいかない：不能~
    重要じゅうような会議かいぎだから，責任せきにん者しゃのわたしが遅おくれていくわけにはいかない。

  • {V / ANな / AI / Nの} わけだ：①解释说明, 因为~②原来如此
    今日の面接では失敗してしまった。準備不足だったわけだ
    暑いわけだ。 38度もある。

だけ

  • {V}だけ{V}：~可能失败(但去试试)
    今から行っても間まに合わないかもしれないけど、行くだけ行ってみようよ。

  • {V可能型 / Vたい / ほしい / 好きな} だけ: 尽可能做~
    悲かなしいときは泣なきたいだけ泣なけばいいよ。

  • {V / AN / AI / N} だけ / だけあって / だけに / だけのことはある: 不愧是~
    彼女かのじょは英語えいごの発音はつおんがとても綺麗きれいで、アメリカへ留学りゅうがくしただけのことはある。

  • {V / ANな / AI / N} だけに: 正因为...更加...
    妹はもともと体が弱いだけに、風邪を引いても心配だ。

  • {V / ANな / AI / N} だけに / だけあって / だけのことはある: 正因为...反倒...
    優勝ゆうしょうすることを期待きたいしていただけに、負まけたときの失望しつぼうは大おおきかった。

  • {V / AN / AI / N} だけの: 足以...的...
    私わたしには自分じぶんで自分じぶんの未来みらいを切きり開ひらくだけの力ちからがある。

次第

  • {Nの} 次第だ/次第で(は): 表示原委、情况
    父ちちが病気びょうきのため、実家じっかの豆腐とうふ屋やを引ひき継ついだ次第しだいだ。

  • {N} 次第だ / 次第で(は)：取决于~, 要看~来定
    頼たのみ方かた次第しだいではあの人ひともこの仕事しごとを手伝てつだってくれるかもしれない。

  • {V连用 / V3する / N} 次第: (现在还做不到,但)...的话马上做..., 次第[しだい]: 顺序
    検査けんさの結果けっかが分わかり次第しだい、ご連絡れんらくいたします。

上

  • [V / AI / ANな / Nの] 以上(は): 既然(因为)...就...
    もう酒さけは飲のまないと決きめた以上いじょうは、どんなに誘さそわれても絶対ぜったいにそれを守まもりたい。

  • {Vる / Vた} 上は：既然(因为)...就...(语气比[以上]强烈), 上[うえ]
    こうなった上うえは、覚悟かくごを決きめて最後さいごまで頑張がんばるしかない。

  • {V / Nの} 上[うえ]で(は) / 上での: 在...上(方面), 后跟重要的事
    仕事の上うえでは別に問題もんだいはない。

  • {Nの} 上[じょう] / 上は / 上も：从在...上，...方面
    生命せいめい倫理りんり上じょうの問題もんだいで技術ぎじゅつに反対はんたいする意見いけんが多おおい。

  • {Vた / Nの} 上で(の)：在...之后
    私わたしが皆様みなさまのご意見いけんを伺うかがった上うえで、来週らいしゅうご報告ほうこくいたします。

  • {V / AI / ANな / Nである / Nの} 上(に): 不仅...而且...(追加)
    台風たいふうが近ちかづいてきて、風かぜが強つよい上うえ、雨あめも激はげしく降ふっている。

つもり

  • {V / Vない} つもりだ: purpose
    今度こんどの日曜日にちようびに遊園地ゆうえんちへ行いくつもりです。

  • {Vた} つもりで: 本来不是这样就当是~
    学生がくせい時代じだい戻もどったつもりで、日本語にほんごを勉強べんきょうする。

  • {Vた} つもりで: 就算是~, 当作~一样
    歌手かしゅになったつもりで、皆みなの前まえで歌うたを歌うたう。

  • {Vた / Vている / ANな / AI / Nの} つもり: 自己认为...(别人不觉得)
    自分じぶんじゃまだまだ若わかいつもりでいるよ。

其他

  • {Vない} ざるを得ない：Have to(消极), 得ない, ざる=>古日语否定
    する⇒ せざるをえない
    2回も同じ間違いをするとは、注意が足りなかったと言いわざるをえない。

  • {Vた / AI / ANな} ことに：倒装用法, 强调心情感情
    残念なことに、行きつけの美容院びよういんが閉店へいてんしてしまった。

  • {N / AN} でも何なんでもない: 根本不~
    {AIく} もなんともない
    そんなの痛いたくも何なんともないよ。

  • {V} を振り出しに: 以...为开端
    彼かれは小学校しょうがっこうの教師きょうしを振ふり出だしに社会しゃかいに出でた。

  • {V} ようでは / ようじゃ: 继续~的话不会有好结果
    上級じょうきゅうになっても、知しらない言葉ことばをいちいち辞書じしょで調しらべているようでは、読解どっかいはうまくならないと先生せんせいに言いわれた。

  • {N / AN / Aい} ぶって: 装作~, ぶる: 装做
    あいつ、先生せんせいの前まえだといい子こぶるけど、掃除そうじサボるし、宿題しゅくだいも誰だれかの写うつしてるんだぜ。

  • {N} という {N}: 全部~(前后名词一致)
    今回こんかいの森林しんりん火災かさいで、この辺へんの木きという木きは、1本いっぽん残のこらず燃もえてしまった。

  • {V} とか: 听说
    え！今日きょうハイキングに行いくの！？テレビで台風たいふうが来くるとか言いってたよ。

  • {N} たる / たるもの: 作为...(应该)...
    選手せんしゅたるもの、試合しあいにおいては堂々どうどうと戦たたかえ。

  • {N} ともあろうものが: 身为...竟然...
    警察けいさつ官かんともあろうものが、強盗ごうとうを働はたらくとは。

  • {V / N} に足る / に足りる: 足以...
      {V / N} に足りない/に足らない: 不足以...
    合格ごうかくと認みとめられるに足たる成績せいせきを示しめさなかった者ものには再さい試験しけんを課かす。
    そんなことは取とるに足たらない。

  • {Vた} ら {Vた} で: 了也就
    母ははによく「部屋へやを片付かたづけなさい」としかられるのだが、片付かたづけたら片付かたづけたで今度こんどは、「片付かたづけられるなら、どうして普段ふだんから片付かたづけないの」と、またしかられる。

  • {N} ごとき: ...等（怎么能够..呢）
    私わたしの気持きもちが、あなたごときにわかるものか。

  • {N} であれ / であろうと: 无论...，都...
    嘘うそをつくとは、どんな理由りゆうであれ、許ゆるされない。

  • {V / N} には及ばない: 不必..., 及およぶ: 达到
    検査けんさでは何なにの異常いじょうも見みつからなかったので、心配しんぱいするには及およばない。

  • {V} 術がない: 无法做到..., 術[じゅつ]: 方法
    ことここに至いたっては、もう成なす術じゅつがない。

  • {V / N} というところだ/といったところだ: 大概是~自分じぶんで料理りょうりを作つくるといっても、せいぜいサラダとか茹ゆで卵たまごといったところだ。

  • {N} までして: 到...的地歩
    自然しぜん破壊はかいまでして、地域ちいきの経済けいざいを発展はってんさせる必要ひつようはない。

  • {N} を余儀なくさせる: 迫使..., 余儀[よぎ]: 其他办法
    両国りょうこく関係かんけいの惡化あっかが、プロジェクトの中止ちゅうしを余儀よぎなくさせた。

  • {Vて / AIくて / ANで} はやりきれない: ...让人受不了, 遣り切れない: 完成不了,受不了
    こう寒さむくてはやりきれない。

  • {N} にあって（は）/ にあっても: 在...之下...
    どのような困難こんなんな状況じょうきょうにあっても、あきらめてはいけない。
